# 🎀 Nuestro Lugar - Espacio Romántico Kawaii

¡Bienvenido/a a nuestro rincón especial en la web! Este proyecto es un sitio web privado, dinámico y muy colorido diseñado con la estética de **Hello Kitty y Sanrio**, lleno de sorpresas interactivas diarias y semanales.

## 🌟 Características Principales
* **Fondo Animado:** Iconos dinámicos y coloridos (moños, gatitos y corazones) flotando por la pantalla.
* **Galleta de la Fortuna:** Una galleta clásica en 3D que se quiebra para revelar un mensaje diario y se bloquea por 24 horas.
* **Calendario de Adviento:** 31 días con misiones y fotos sorpresa. ¡El día 23 está marcado como especial de aniversario! (Las fotos de días pasados se ocultan automáticamente para mantener el secreto el próximo mes).
* **Rasca y Gana Semanal:** Cupones interactivos para raspar la pantalla con un temporizador de bloqueo de 7 días.

## 🛠️ Tecnologías Utilizadas
* HTML5 y CSS3 Personalizado (Estilo Pop/Sticker)
* Bootstrap 5 (Estructura y Ventanas Modales)
* JavaScript Vanilla (Control de tiempos con `localStorage` y Canvas para el raspadito)
* FontAwesome (Iconos flotantes)

## 📸 Configuración de Imágenes
Para que el calendario funcione correctamente, asegúrate de subir tus fotos a la carpeta `/imagenes/fotos/` con los nombres del `1.png` al `31.png`, además de `aniversario.png`.
